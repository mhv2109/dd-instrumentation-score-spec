**Rule ID:** RES-001

**Description:** `host` is present.

**Rationale:** The `host` uniquely identifies a resource, and can be used as the process identifier without taking other resource attributes into account. In Datadog, the `host` attribute represents the originating host or instance (e.g., AWS EC2 instance ID).

**Target:** Resource

**Criteria:** The `host` resource attribute MUST appear in the resource.

**Impact:** Normal
