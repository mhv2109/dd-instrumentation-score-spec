**Rule ID:** MET-006

**Description:** Metric names do not equal semantic convention attribute keys.

**Rationale:** A metric name that exactly matches an attribute key defined in the semantic conventions (e.g., `http.status_code`, a span-level attribute) causes confusion and should be avoided.

**Target:** Metric

**Criteria:** Metric names MUST NOT equal attribute keys specified in the semantic conventions.

**Impact:** Low
